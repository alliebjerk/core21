# The AI Agent Blueprint — The Core 21

Twenty-one AI agents across six departments, plus an Onboarding Agent to get you started.

**This is a paid product.** If you haven't purchased The AI Agent Blueprint, visit [alliebjerk.com](https://alliebjerk.com) to learn more.

## Quick Start

1. Fork this repo (click "Fork" on GitHub — make your fork **private**)
2. Clone your fork to your computer
3. Open your Notion workspace (link provided with your purchase)
4. Follow the Setup Guide in Notion
5. Every file is pre-filled with a worked example from a real business. Replace the example content with your own business specifics as you install each agent. (Start with voice.md and business.md at the root — every agent reads those.)
6. Run the Onboarding Agent first — it fills in your voice.md and business.md and recommends your first three agents

## What's Inside

Each numbered folder is one agent with template files to customize for your business. Documentation, training videos, and step-by-step guidance live in the Notion workspace (not in this repo).

## Credentials

Your `.env` file and `.mcp.json` files are in `.gitignore`. They never get pushed to GitHub. Keep your API keys on your local machine only.
